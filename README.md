# Network Packet Analysis Lab

This project provides a basic framework for network packet analysis using Python. It utilizes the `scapy` library to capture and analyze network packets.

## Requirements

- Python 3.6+
- `scapy` library

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/network-packet-analysis.git
   cd network-packet-analysis
   ```

2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

Run the main script to start capturing and analyzing packets:
```bash
python packet_analysis.py
```
